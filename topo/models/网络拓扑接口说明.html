1、流量统计柱图接口

说明：点击流量统计按钮弹出的柱状图数据

	{
		"ChinaNet":100,            /*电信流量*/
		"CMCC":100,                /*移动流量*/
		"LT":100,                  /*联通流量*/
		"fireGate":100,            /*防火墙流量*/
		"gaNet":50,                /*公安网流量*/
		"vpn":100,                 /*VPN流量*/
		"chengdu":100,             /*成都市局流量*/
		"infocenter":100,          /*信息中心流量*/
		"anxun":100                /*安讯机房流量*/
  }

以上分类仅作展示用，柱状图X、Y轴数据需要后台提供，样式如下：

  	    [{"name":"中国移动","flow":200},
             ...
             ]

2、系统状态概要接口

说明：标题下面第二行数据

 {
		"userNum":100,            /*终端用户数*/
		"runTime":"96:54:33",     /*系统运行时长*/
		"station":"正常",         /*平台运行状态*/
		"requests":600	          /*应用访问量*/
}

3、拓扑图接口

拓扑节点数据明细

需要提供拓扑节点数据样式如下：

{
  "success":"",
  "error":"",
  "data":[{
    "name": "防火墙",             /*节点名称*/
    "icon":"images/firewall.png", /*图标url*/
    "collapse":false,             /*是否展开(可不提供)*/
	"ll":500,                    /*该节点到父节点的链路上的流量(根节点没有这项,除根节点外其他节点都需提供)*/
    "children":[
	{...,
	   children[],               /*子节点中还可以拥有子节点，层级不定*/
	...},
	...
	]          
   }]
}
